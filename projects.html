Learning Journal PWA – Lab 4

Extended the Learning Journal PWA with advanced interactivity using three types of APIs:

  • Storage API: Used localStorage to save and display journal entries persistently.
  • Browser API: Integrated the Notification API to alert users when entries are saved or deleted.
  • Third-Party API: Embedded a YouTube coding tutorial dynamically using JavaScript and iframe.

All features were modularized into separate JavaScript files and styled to match the journal’s responsive layout and dark mode support.

Technologies: JavaScript, Storage API, Notification API, YouTube Embed, Modular JS
