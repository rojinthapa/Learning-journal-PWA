Learning Journal PWA – Lab 7

Transformed the Learning Journal into a fully installable Progressive Web App (PWA) with offline capabilities:

  • App Manifest: Added manifest.json to enable "Add to Home Screen" functionality with custom icons and theme colors.
  • Service Worker: Implemented a Service Worker (sw.js) to intercept network requests and manage caching.
  • Offline Support: Used "Network First" strategy for dynamic data (API) and "Cache First" for static assets to ensure the app works without internet.
  • Extra Feature: Built a real-time offline detection system that notifies users via a banner when they lose connectivity.

Technologies: Service Worker API, Cache Storage API, Web Manifest, JavaScript, Flask
